IDS Server (ids.r2dfoo) Release Notes

3.0.0

  • Update storaged-ids-plugin to 2.0.0, which has an updated StorageD interface and an updated protobuf library.
  • Update other dependencies.
  • Update test environment settings to port 8080 to avoid complications caused by HTTPS certificates.

2.0.1

Upgrade storaged-ids-plugin to 1.3.0, which fixes incompatibility of logback versions.

2.0.0

  • Upgrade from JavaEE to JakartaEE 10. Requires Java 11+ and an application server that supports JakartaEE 10 such as Payara 6.
  • Upgrade logback-classic to 1.2.12

1.0.0

First release of ids.r2dfoo. (Versions of the forked ids.server already exist up to 1.11.0).
